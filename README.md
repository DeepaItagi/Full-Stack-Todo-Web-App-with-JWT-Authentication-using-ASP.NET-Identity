# Full-Stack-Todo-Web-App-with-JWT-Authentication-using-ASP.NET-Identity
This is a Todo Web App with CRUD operations &amp; user login/account management. Once registered successfully, individual users can login, add, edit, delete &amp; track their respective Todo-items. Technology Stack used: PostgreSQL, ASP.NET Core Web Api, Angular, Entity Framework Core(Used Code First Approach).
